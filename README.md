Un script en Bash que automatiza la descarga, compilación e instalación de los comandos pk2cmd y la interfaz QPickit.
Fue probado en Linux Mint 22.2. (Debería funcionar en Ubuntu 22.04/24.04 y derivados de Debian)

Instrucciones de uso:

Bash

chmod +x PK2_Install_Script.sh

sudo sh ./PK2_Install_Script.sh

Este script instala herramientas desarrolladas por terceros: pk2cmd de jaka-fi y QPickit de GTRONICK.

https://github.com/GTRONICK/QPickit

https://github.com/jaka-fi/pk2cmd

Si deseas eliminar el software completamente, ejecuta:

sudo sh ./PK2_Uninstall_Script.sh

PicKit2 y un derivado MicroProg R2 basado en el PicKit2 fueron exitosamente probados.
